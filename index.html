현장 검증 필수. 공개 정보 기반의 검증된 시작용 디렉터리입니다. 가격대(예산/중급/프리미엄/럭셔리)는 시장 포지셔닝 추정치이며 견적이 아닙니다. 실제 단가·트레이드 할인율·리드타임·MOQ·결제조건·담당자 직통은 직접 컨택해 채우세요. 직접 추가 = 사용자가 제공해 검증된 업체 · 🚩 = 정확도 재확인 필요.

직거래·제조 = 직수입/자체 제조(단일 마진) · 유통·재가공 = 사와서 재가공·재판매(마진 이중, 주의) · 확인 = 공급구조 불명. (추정)은 설명 기반 자동판별이라 거래 전 직접 확인 권장. 상단 공급구조 필터로 걸러 보세요.

후기 교차검증: 요즘 업체 홈페이지는 AI 검색에 잘 잡히도록 세팅(AEO)된 경우가 많아 자사 설명만 믿으면 안 됩니다 — Google 지도 평점·리뷰, Trustpilot 등 실사용 후기를 함께 확인하세요. 카드의 G★표기 = 실제 Google 평점. 지도 링크는 정확한 장소로 연결되며, 쇼룸/사무실이 다르면 위치 링크가 따로 표시됩니다.

각 카드의 진행 상태(미정→컨택→방문→견적비교중→확정/제외)를 선택하고 담당자·연락처·견적가·자료 링크·메모를 기록하세요 — 상단·사이드바 진행현황과 지역/상태 필터에 반영되고 자동 저장됩니다. 팀 공유·기기 변경은 상단 드라이브 동기화(자동) 또는 백업 내보내기 / 불러오기(수동)를 쓰세요.

공통 — 어느 업체든 이 3가지부터

  • ① 있나? — 지정 자재(또는 비슷한 무늬·색)를 취급하는지 · 실물/재고 확인 가능한지
  • ② 대안은? — 단종/납기 길면 유사 대체 후보. (지정 + 대안 2안 견적 요청)
